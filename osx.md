## Basic tools

```sh
brew install vim tmux fish git
```

## Keyboard

- Install [Karabiner](https://pqrs.org/osx/karabiner/)
- Install [Seil](https://pqrs.org/osx/karabiner/seil.html) to control what Caps Lock does
